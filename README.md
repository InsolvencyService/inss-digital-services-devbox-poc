# inss-digital-services-devbox-poc

DevBox definitions for a proof of concept devbox.

## Repository Structure

- **image_definitions/**
	- Contains production-ready image definition YAML files for Dev Box environments.
	- See [image_definitions/README.md](image_definitions/README.md) for details on each image definition and folder structure.

- **experimental_image_definitions/**
	- Contains experimental or in-progress image definition YAML files and prototypes.
	- See [experimental_image_definitions/README.md](experimental_image_definitions/README.md) for details and examples.

- **user_customization_files/**
	- Contains user customization YAML files for additional Dev Box setup and configuration.
	- See [user_customization_files/README.md](user_customization_files/README.md) for usage instructions and examples.

---

Each folder above contains its own README with more information about its contents and usage.
